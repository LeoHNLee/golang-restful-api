# Building RESTful APIs with Go

## API Outline

- Resource: a list of users
- List item data: name, role, and id
- Functionality: CRUD
- Endpoints:
    - collection: /users
    - item: /users/{id}
- data format: JSON

### create item: POST

### access the collection: GET /users

### Access the Item: GET /users/{id}

### Replace an Item: Put /users/{id}
